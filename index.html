Michael Noi: WEBD-225

This is my personal site for the WEBD-225 course. My name is Michael Noi and I am a final-year cybersecurity student at Champlain College.

Week 2

In Week 2, I gained insight into CSS preprocessors like Sass and SCSS, and how they can streamline styling by using tools such as variables, nesting, and mixins. These features make CSS more maintainable and reduce repetitive code. I supplemented my learning with documentation and YouTube videos to better understand advanced concepts like maps and loops, which I applied in the CodePen assignment. I also learned about Git workflows and specifically branching and merging—which allow developers to work on separate tasks simultaneously and then integrate their changes seamlessly into the main project.
